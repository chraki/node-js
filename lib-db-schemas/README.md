## Migrations

### 1)Todo(To create tables in database)
```bash
cd lib-db-schemas/
npm install
npm run migrate
```

### 2)To Undo(To drop tables in database)
```bash
cd lib-db-schemas/
npm install
npm run migrate:undo
```
